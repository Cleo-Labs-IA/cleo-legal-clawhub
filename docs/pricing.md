## Composite endpoints

`/cleo-compliance` and the `compliance_check` MCP tool consume **5 quota units per call** (they do the work of 5 lookups).

Bulk endpoints (`search_legal_bulk`, `get_documents_bulk`) consume **N units** where N = array length.
